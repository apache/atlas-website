TypesResource Resource

This class provides RESTful API for Types. A type is the description of any representable item; e.g. a Hive table You could represent any meta model representing any domain using these types.

GET /types

This method has been deprecated.

Return the list of type names in the type system which match the specified filter.

Request Parameters
name type description
notsupertype query returns types which do not contain the given supertype Its possible to specify combination of these filters in one request and the conditions are combined with AND For example, typeCategory = TRAIT && supertype contains 'X' && supertype !contains 'Y' If there is no filter, all the types are returned
supertype query returns types which contain the given supertype
type query returns types whose relationshipCategory is the given typeCategory
Response Body
media type data type description
application/json;charset=UTF-8 (custom) list of type names

Example

Request
GET /types
Accept: application/json;charset=UTF-8
Response
HTTP/1.1 200 OK
Content-Type: application/json;charset=UTF-8
